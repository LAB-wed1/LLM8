ฉันได้สร้างโจทย์สอบ React Native 5 ข้อพร้อมโค้ดตัวอย่างเรียบร้อยแล้ว ดังนี้:
ไฟล์ task1-product-list.js - แอปแสดงรายการสินค้าพร้อมฟังก์ชันค้นหา
ไฟล์ task2-todo-app.js - แอป Todo พร้อมการจัดเก็บข้อมูลใน AsyncStorage
ไฟล์ task3-firebase-auth.js - ระบบยืนยันตัวตนด้วย Firebase Authentication
ไฟล์ task4-navigation-app.js - แอปหลายหน้าที่ใช้ React Navigation
ไฟล์ task5-firestore-products.js - แอปจัดการสินค้าด้วย Firebase Firestore
นอกจากนี้ยังมีไฟล์ exam-tasks.md ที่รวมคำอธิบายโจทย์ทั้ง 5 ข้อพร้อมคำใบ้สำหรับการทำข้อสอบ
ทุกไฟล์มีโค้ดตัวอย่างที่ทำงานได้จริง ซึ่งคุณสามารถใช้เป็นเฉลยหรือแนวทางในการตรวจคำตอบได้ โจทย์ทั้งหมดครอบคลุมความรู้หลักของ React Native ตั้งแต่พื้นฐานไปจนถึงการใช้งานกับบริการภายนอกอย่าง Firebase
